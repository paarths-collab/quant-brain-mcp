🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

"GEMINI_API_KEY not found"
  → Set: export GEMINI_API_KEY="your-key"
  → Or add to .env file

"Import error"
  → Run: pip install -r requirements.txt

"Rate limit exceeded"
  → Add caching or upgrade Gemini plan (still cheap)

"Wrong market data"
  → Change symbols in SmartStockSelector
